- tmux plugin management: https://github.com/tmux-plugins/tpm

## Tmux command

| #   | Command | Action                          |
| --- | ------- | ------------------------------- |
| 1   | ` c     | Create a new window             |
| 2   | ` ,     | Rename current window           |
| 3   | ` &     | Close current window            |
| 4   | ` w     | List windows                    |
| 5   | ` 0..9  | Switch window from index 0 .. 9 |
| 6   | ` l     | Toggle last active window       |
|     |         |                                 |
|     |         |                                 |
|     |         |                                 |
